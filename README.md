# Product Store Web App

A React and Vite e-commerce web application that allows users to browse products, view product details,and navigate through a modern store interface.


## Features

-HomePage
-Product Listing Page
-User signup and login pages
-Navigation bar and footer
-Built with React and Vite

## Technologies Used

-React
-Vite
-JavaScript
-CSS
